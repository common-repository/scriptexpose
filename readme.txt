==  Description  ==

The installation creates an ScriptEt  menu dedicated 
you can build modules , where you can place text and images , and assign it to template
through the ScriptEt menu you find a panel to customize the template to combine the modules.
Finally it will generate the shortcode to be included in html pages or in the articles of your wordpress
In addition, each module can contain more than one items
which we will be listed in the articles in which you include the shortcode dedicated .

ScripExpose contains third party software .
jquery.colopicker with which you can change colors and transparencies of background

== Installation ==

1. Upload `ScriptEt.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
